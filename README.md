\# AI-Powered Resume Screening \& Candidate Ranking System 📄🚀



A full-stack application that leverages Natural Language Processing (NLP) to automate the recruitment process by ranking resumes against job descriptions using semantic similarity.



\## 🌟 Key Features

\- \*\*Semantic Matching:\*\* Uses \*\*Sentence-BERT (SBERT)\*\* to understand the context of resumes, going beyond simple keyword matching.

\- \*\*Automated PDF Parsing:\*\* High-fidelity text extraction from unstructured PDF files.

\- \*\*FastAPI Backend:\*\* High-performance, asynchronous API providing real-time match scoring.

\- \*\*React Dashboard:\*\* User-friendly interface for recruiters to upload multiple resumes and view ranked results instantly.



\## 🛠️ Tech Stack

\- \*\*Languages:\*\* Python, JavaScript

\- \*\*Backend Framework:\*\* FastAPI

\- \*\*AI/ML Model:\*\* Sentence-Transformers (`all-MiniLM-L6-v2`)

\- \*\*Frontend Framework:\*\* React.js

\- \*\*Mathematical Logic:\*\* Cosine Similarity



\## 🚀 How to Run Locally



\### 1. Backend Setup

1\. Create a virtual environment: `python -m venv venv`

2\. Activate it: `.\\venv\\Scripts\\activate` (Windows)

3\. Install dependencies: `pip install fastapi uvicorn sentence-transformers PyMuPDF`

4\. Run the server: `python main.py`



\### 2. Frontend Setup

1\. Navigate to folder: `cd frontend`

2\. Install dependencies: `npm install`

3\. Start the UI: `npm start`
